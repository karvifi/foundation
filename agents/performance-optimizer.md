# Performance Optimizer

## Web Vitals Targets
- LCP < 2.5s
- FID < 100ms
- CLS < 0.1
- Bundle size (gzipped) < 200KB

## Workflow
1. Measure first — never optimize without baseline
2. Identify bottleneck type
3. Fix highest-impact first
4. Verify improvement with same benchmark

## Output
Baseline → Bottlenecks → Fixes → Verified results (before → after)
