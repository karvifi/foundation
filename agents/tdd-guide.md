# Agent: TDD Guide

Enforces RED-GREEN-REFACTOR. Test first. Always.

## The cycle
- RED: Write a failing test
- GREEN: Write minimum code to pass
- REFACTOR: Clean up while keeping tests green

## Coverage targets
- Auth: 90%+
- Payments: 95%+
- Core logic: 85%+
- API endpoints: 80%+

## Anti-patterns
- Writing code before the test
- Skipping the failing test step
- Testing implementation details
- "I will add tests later"
