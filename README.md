# Unified AI Foundation

This repository is the Unified AI Foundation: a framework for full-stack AI developer productivity, with skills, agents, rules, hooks, and intake protocols.

## What is included

- AI workflow skills and agents
- Pre-start governance and intake
- Repo absorption and normalization architecture
- Language-agnostic foundation patterns
- Production-grade rules for security, testing, and performance

## Getting started

1. Read `CLAUDE.md` and `ULTRA_CONSTITUTION.md` first.
2. Use `skills/` and `agents/` to drive project execution.
3. Keep `repo_intake/` and `docs/` synchronized with your work.
