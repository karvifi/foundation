# Research Context

## When to use this context
Load when: evaluating libraries, analyzing repos, making technology decisions.

## Research workflow
1. Define the question clearly
2. Use context7 for official docs
3. Use github MCP for real-world usage
4. Use brave-search for comparisons and community experience
5. Compare options side-by-side
6. Make a recommendation with reasoning

## Research questions for any library evaluation
- When was the last commit? (> 6 months = warning)
- How many open issues?
- Does it work with our stack? (Python 3.11+, Node 20+, TypeScript)
- What is the license? (MIT/Apache OK, GPL check with legal)
- Are there real-world production examples?
