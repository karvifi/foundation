## 🔐 SECURITY — ABSOLUTE MANDATES

These apply to EVERY project, EVERY time, with ZERO exceptions:

```
1. NEVER hardcode any secret, key, token, password in source code
2. NEVER commit .env files
3. ALWAYS parameterize SQL (no f-strings, no string concatenation)
4. ALWAYS validate ALL user input (Pydantic/Zod at every boundary)
5. ALWAYS authenticate AND authorize (both, not just one)
6. ALWAYS set max_tokens on EVERY LLM call
7. ALWAYS rate-limit AI/LLM endpoints
8. ALWAYS run security scan before first production deploy
9. NEVER log PII from prompts or user messages
10. ALWAYS wrap user input in LLM prompts: <user_input>...</user_input>
```
