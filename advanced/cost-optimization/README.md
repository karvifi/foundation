# Cost Optimization Tools

Calculate, monitor, and optimize infrastructure + AI costs.

## Tools Included
- `cost-calculator.py` - Calculate monthly costs  
- `optimizer.py` - Find cost savings opportunities
- `budget-alerts.yaml` - Alert when approaching budget
- `rightsizing.py` - Right-size infrastructure

## Usage
```bash
python3 cost-calculator.py --users 10000 --ai-calls 50000
# Output: Detailed cost breakdown + optimization recommendations
```
