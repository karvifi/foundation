# SKILL: SEO Optimization

## Purpose
Make content and websites discoverable and rank well in search — sustainable, not manipulative.

## Three Pillars

### Pillar 1: Technical SEO
```
□ Page speed: Core Web Vitals (LCP < 2.5s, FID < 100ms, CLS < 0.1)
□ Mobile-first responsive design
□ HTTPS everywhere
□ Canonical tags on all pages
□ XML sitemap created and submitted
□ robots.txt configured correctly
□ Structured data (Schema.org) for content type
□ No broken links (audit with Screaming Frog or similar)
□ Proper heading hierarchy (one H1 per page)
□ Meta titles (50-60 chars) and descriptions (150-160 chars)
□ Image optimization (WebP, compressed, descriptive alt text)
□ Internal linking structure (every page reachable in 3 clicks)
□ Pagination handled correctly (rel=next/prev or canonical)
□ hreflang for multi-language sites
```

### Pillar 2: Content SEO
Keyword Research Process:
1. Identify seed keywords from user's business/topic
2. Find long-tail variations (3-4 word phrases with clear intent)
3. Check search volume + keyword difficulty (use Ahrefs, SEMrush, or free alternatives)
4. Classify intent: Informational / Navigational / Commercial / Transactional
5. Map keywords to pages (one primary keyword per page)

Content Structure Template:
```markdown
# [Primary Keyword] — [Additional Modifier]
(H1 — include primary keyword, max 60 chars)

[Meta description: 150-160 chars, include keyword, include value proposition]

## Introduction (100-150 words)
- Hook + primary keyword in first 100 words
- State what the reader will learn

## [Subtopic H2] (keyword-rich but natural)
### [Sub-subtopic H3]
...

## Frequently Asked Questions
[Target featured snippets — answer each in 40-60 words]

## Conclusion
[CTA + next step]
```

### Pillar 3: Local/Geo SEO (if applicable)
```
□ Google Business Profile (GBP) complete and verified
□ NAP (Name, Address, Phone) consistent everywhere
□ Local schema markup (LocalBusiness)
□ City/region keywords in title tags, H1, content
□ Location-specific landing pages (one per service area)
□ Local link building (chambers of commerce, directories, partnerships)
□ Review generation strategy (GMB, Yelp, Trustpilot)
□ Geo-targeted Google Ads supporting organic effort
```

## SEO Audit Process (for existing sites)
1. Crawl site → find technical errors
2. Check Google Search Console → find indexing issues
3. Analyze top pages → what is already working?
4. Find quick wins → thin content to expand, missing meta tags
5. Competitive gap analysis → keywords competitors rank for, you do not
6. Prioritize by impact/effort matrix

## AI Search Optimization (2025+)
```
□ Answer questions directly and clearly (AI pulls answers, not pages)
□ Use structured content (bullets, headers, tables)
□ Include expert quotes and citations
□ Target "zero-click" queries with value (still builds brand)
□ Optimize for voice search (conversational phrasing, FAQs)
□ Ensure E-E-A-T: Experience, Expertise, Authority, Trust signals
```
