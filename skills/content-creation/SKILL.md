# SKILL: Content Creation

## Purpose
Create high-quality content that serves a specific goal for a specific audience.

## Content Types & Templates

### Blog Post / Article
```
Title: [Primary keyword] — [benefit or curiosity hook] (50-60 chars)
Meta: [150-160 chars with keyword and value promise]

Structure:
H1: [Title]
Intro (150 words): Hook + problem + what you'll learn + credibility signal
H2: [Main section 1]
  H3: [Sub-point]
H2: [Main section 2]
H2: [Main section 3]
H2: FAQ (target featured snippets — 3-5 questions, 40-60 word answers)
Conclusion: Summary + CTA + next step
```

### Email Newsletter
```
Subject line (max 50 chars): [Curiosity/benefit/urgency — test 3 variants]
Preview text (max 100 chars): [Extend the subject, add context]

Structure:
- Hook (first line — what is in it for the reader?)
- 1 main insight or story (200-400 words max)
- 1 actionable takeaway
- 1 CTA (ONE, not three)
- P.S. (often the most read part — use it for secondary CTA)
```

### Landing Page
```
Above the fold:
  H1: [Outcome the customer gets] (not what you sell)
  Sub: [How you deliver it] or [Who it's for]
  CTA: [Action verb] + [immediate benefit]
  Social proof: [Number or name]

Below the fold:
  Problem section (agitate the pain)
  Solution section (your approach)
  Features → Benefits (for each feature, ask "so what?")
  Social proof (testimonials, case studies, logos)
  FAQ (handle top 5 objections)
  Final CTA
```

### Social Media Post (platform-specific)
```
LinkedIn:
  Hook line (stops the scroll — no "I am excited to share")
  1-3 short paragraphs
  Key insight or lesson
  1 question or CTA
  3-5 relevant hashtags

Twitter/X:
  Strong first tweet (hook)
  Thread if >280 chars
  End with summary + engagement prompt

Instagram:
  First line is the hook (visible before "more")
  Story or value in body
  CTA at end
  Hashtags: 5-10 targeted (not generic)
```

### Video Script
```
Hook (0-5 seconds): State the ONE thing viewer will get
Problem (5-30 seconds): Why this matters to them
Promise (30-60 seconds): What they will learn
Content delivery: Main points with examples
CTA: One specific next action
Thumbnail concept: [visual + text overlay]
```

## Quality Standards
- Lead with the reader's/viewer's benefit, not yours
- Every paragraph earns the right to the next
- One idea per paragraph
- CTA is specific (not "let me know what you think")
- Edited for half the original word count (ruthless cutting)

## Quality checks
- [ ] Opening hook stops the scroll/skip
- [ ] Content delivers the promise made in title/subject
- [ ] One clear CTA (not multiple competing ones)
- [ ] Cut all filler words (very, really, basically, just)
- [ ] Read aloud test passed (natural language, no jargon)
